# ✨Tecnologias
* Node
* TypeScript
* Express
* TypeORM
* SQLite
* Jest
* Swagger

# 🚀 Como Executar
* Clone o repositório
* Instale as dependências (npm install)
* Inicie o servidor (npm run dev)

Agora você pode acessar localhost:3000/docs no seu navegador para ter acesso a documentação da API.
